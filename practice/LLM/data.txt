Q: I uploaded a virus by mistake. What happens? A: Our servers run on secure Linux environments with restricted permissions. Uploaded files are treated as static binary data and are never "executed" on our server. This prevents malware from running. The file will simply be deleted during the hourly cleanup cycle.

Q: Can I convert password-protected PDFs? A: For security reasons, we currently do not support cracking or bypassing PDF passwords. You must provide the password to unlock the file before we can process it, or upload an unlocked version.

Q: My download was blocked by Chrome/Edge. A: Browsers sometimes flag downloads from new websites. If you see a warning, you can usually click "Keep" or "More Info > Run Anyway." We guarantee that files generated by ToolsHub are clean documents and contain no executables or scripts.
